# HK 工伤赔偿系统现代化改造

## 项目定位

面向香港工伤赔偿业务的现代化改造项目，核心是将历史 Struts/JSP/Oracle 体系逐步迁移到 React + Spring Boot + PostgreSQL/EDB 架构。我的价值重点在于迁移断点分析、联调验证、测试补强和风险识别。

## 技术关键词

- Java 21
- Spring Boot 3
- Spring Security
- MyBatis
- Oracle
- EDB/PostgreSQL
- React 19
- TypeScript
- Vite
- MUI
- DMS

## 我负责的内容

- 参与 ECCS/PMCOS 现代化改造中的重点模块排障和迁移分析
- 支持 FCO、CP01、CP09、DMS 等模块的问题定位、修复验证和联调
- 对高风险保存链路补充测试和校验，降低迁移回归风险
- 输出新旧系统差异分析、阶段进度和专项问题材料

## 可写进简历的亮点

- 通过旧系统 JSP/Action 与新系统 React/Spring Boot 逐层对比，定位 CP09 二次授权闭环缺失问题
- 对 FCO/CP01 等高风险保存链路做测试补强和校验完善，提升迁移稳定性
- 处理 DMS 元数据、日期兼容和非案件查询等前后端联调问题
- 用进度、统计和专项分析材料提高迁移项目的沟通效率与风险可见性

## 简历短版

参与 ECCS/PMCOS 现代化改造，负责 FCO/CP01/CP09/DMS 等模块的问题分析、测试补强和前后端联调，输出新旧系统差异分析与阶段性进度材料，支持核心业务迁移质量提升。

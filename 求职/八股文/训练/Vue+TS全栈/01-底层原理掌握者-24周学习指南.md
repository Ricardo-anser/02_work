# 底层原理掌握者：Vue3 + TypeScript + Node.js 24 周学习指南

## 00. 使用说明

- 路线目标：把“会用”升级为“能解释、能拆解、能复现、能迁移”。
- 训练原则：运行时优先于背诵，实验优先于结论，最小实现优先于空泛总结。
- 模块约束：Node.js 后端代码必须显式选用 `CommonJS` 或 `ESM`，不允许混用。
- 工程约束：不把商业 UI 框架作为起点；需要状态管理时，以 Pinia 的官方心智模型为参照，而不是从 API 表层记忆入手。

## 01. 前置知识

### 必须掌握

- JavaScript Promise、闭包、原型链、`this` 基本语义。
- 浏览器事件循环与 Node.js 脚本执行的基础差异。
- 能独立运行 `node`、`npm` 或 `pnpm` 的基本命令。

### 训练中补齐

- TypeScript 基本注解与接口写法。
- SQL 约束、索引与事务直觉。
- HTTP 请求/响应报文的基础结构。

### 暂时忽略

- 复杂 UI 美化与动画。
- 大而全的后端框架封装。
- 提前优化微前端或分布式系统。
